# Информация о программе
Java (JDK) 17.0.15
Система сброки Maven 3.9.9
Сторонние библиотеки не используются

# Инструкция по запуску утилиты FilterUtil
Собранный исполняемый файл `FilterUtil.jar` находится в папке `out`.
Для запуска утилиты необходим установленный JDK версии 17 или выше.
Порядок запуска:
1. Скачать файл `FilterUtil.jar` и сохранить его в любой удобной папке
2. Открыть командную строку
3. Перейти в папку с утилитой
4. Выполнить команду `java -jar FilterUtil.jar [опции]`
После выполнения программа создаст файлы с результатами в указанной директории.

## Поддерживаемые опции:
| Ключ           | Описание                                                                             |
| -------------- | ------------------------------------------------------------------------------------ |
| `-o [путь]`    | Указывает путь к выходным файлам. По умолчанию файлы создаются в текущей директории. |
| `-p [префикс]` | Добавляет указанный префикс к именам выходных файлов.                                |
| `-a`           | Добавляет данные в выходные файлы вместо их перезаписи.                              |
| `-s`           | Включает **краткую статистику** (общее количество элементов каждого типа).           |
| `-f`           | Включает **полную статистику** (максимумы, минимумы, средние значения).              |
| `[файлы]`      | Один или несколько входных файлов для обработки.                                     |


## Пример использования:
java -jar FilterUtil.jar -s -a -p sample- in1.txt in2.txt

## Формат выходных файлов:
integers.txt — список целых чисел
floats.txt — список вещественных чисел
strings.txt — остальные строки
Выходные файлы создаются только если будут содержать хотя бы один элемент.
